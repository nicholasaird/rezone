# Rezone

## Description
An open-ended city simulator where you control the zoning and watch the city grow!

## Compilation

### Ubuntu
Build the executables:
```shell
# Inside the project root
mkdir build
cd build
cmake ..
make
```

Run the tests:
```shell
# Inside /build
./test
```

Run the game:
```shell
# Inside /build
./rezone
```
